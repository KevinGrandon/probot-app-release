> a GitHub App built with [probot](https://github.com/probot/probot) that creates GitHub releases

## Setup

```
# Install dependencies
npm install

# Run the bot
npm start
```

See [docs/deploy.md](docs/deploy.md) if you would like to run your own instance of this app.
